# Exempelwebbplats - HTML & CSS

Detta är en enkel webbplats skapad för att demonstrera hur man bygger en sammanhängande sajt med HTML och CSS.

## Innehåll
*   **HTML:** Semantisk struktur med header, nav, main och footer.
*   **CSS:** En global stilmall (`style.css`) för design och layout.
*   **Responsivitet:** Anpassad för både mobil och desktop.

## Hur man använder detta
1.  Ladda ner eller klona detta repository.
2.  Öppna `index.html` i din webbläsare för att se resultatet.
3.  Utforska koden i `index.html`, `om.html`, `kontakt.html` och `lektion.html` för att se hur sidorna hänger ihop.
